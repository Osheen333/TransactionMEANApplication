# MEAN-project


## Recommended IDE Setup

[VSCode](https://code.visualstudio.com/) 


## Project Setup

```sh
npm install
```
### Compile and Hot-Reload for Server (Node)

```sh
cd server
npm start
```


### Compile and Hot-Reload for Client (Angular)

```sh
cd client
npm start
```

### Compile and Minify for Production

```sh
npm run build
```
